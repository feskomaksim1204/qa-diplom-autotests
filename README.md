# QA Диплом: от модели до тестового фреймворка

## Описание проекта

Это дипломный проект по автоматизации тестирования. Он построен как «траектория роста»: от простой Java-модели бургерной до полноценного тестового фреймворка, покрывающего UI и API.

**Какую проблему решает проект?**  
Показывает, что я умею не просто писать автотесты, а проектировать тестовую архитектуру:  
- сначала написать бизнес-логику (модель бургерной)  
- затем покрыть её API-тестами (REST Assured)  
- и в итоге построить UI + API фреймворк с Selenium, Page Object и Allure

**Стек технологий:**  
- Java 11  
- JUnit 4  
- Selenium WebDriver  
- REST Assured  
- Maven  
- Allure  
- Page Object Pattern

## Системные требования и запуск

**Требования:**  
- Java 11 или выше  
- Maven 3.9+  
- Браузеры: Chrome (последняя версия) или Яндекс Браузер

**Запуск API-тестов заказов (папка `02-api-order-tests`):**

    cd 02-api-order-tests
    mvn clean test
    mvn allure:report

**Запуск UI + API автотестов (папка `03-ui-stellar-burgers`):**

    cd 03-ui-stellar-burgers
    mvn clean test
    mvn clean test -Dbrowser=yandex
    mvn allure:report
